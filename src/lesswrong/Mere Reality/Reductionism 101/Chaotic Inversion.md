# 关于混沌的倒置思考

原文：[Chaotic Inversion](https://www.readthesequences.com/Chaotic-Inversion)

❦ 

最近我和几位朋友聊起了按小时规划工作效率和维持意志力续航的话题——这也是我毕生都在试图攻克的课题。

最近我和几个朋友聊起了有关「按小时规划工作效率和维持意志力续航」的话题——这也是我毕生都在试图攻克的一个课题。（语言更日常一点，信息密度低一点的版本）


我可以做到[在初步遇到难题时不逃跑](https://www.readthesequences.com/OnDoingTheImpossible)（在秒的时间尺度内坚持），也可以做到在同一个问题上持续钻研数年；但是，在*小时*的时间尺度上持续工作，对我来说，却始终像是一场无休止的战役。不用说，相关的行动建议我早已读了一堆又一堆；我从这些建议中获得的最大帮助，就是意识到：其实有相当部分的其他创意工作者也面临着同样的问题，并且也无法战胜它，不论他们推崇的那些建议听起来多么的合理。

朋友们问我：「当你没法工作时，你会做些什么？」（以下对话仅为粗略还原，非现场原话）

我回答说，我通常会随便浏览一些网站，或者看个短视频。 

「其实吧，」他们说，「如果你知道自己有一阵子没法工作，那你应该去看部电影什么的。」 

「问题就在这里，」我回答说，「我必须去做那种短时间内就能完成的事情，比如随便上上网或者看看短视频，因为我随时都可能重新投入工作，而我没法预测具体什么时候——」 

说到这儿我停了下来，因为我突然明白了什么——

我一直把自己的工作周期视为某种**混沌的**东西，某种**不可预测的**东西。虽然我从来没有这么形容，但在心理和行动上，我一直是这么理解和**对待**它的。

可是在这里，我的朋友们似乎在暗示——多么奇怪的想法——**其他**人竟然可以预测自己何时能够重新投入工作，并且据此安排自己的时间。

我第一次想到，我也许是在日常生活这种最普通不过的场景里——而不是在某种高度抽象的地方——犯下了那个该死的老掉牙错误：[心智投射谬误](https://www.readthesequences.com/MindProjectionFallacy)。 
思维投射谬误

我第一次意识到，我可能一直在犯那个该死的陈词滥调——“心灵投射谬误”-，而且不是在高度抽象的领域，就在我平凡的日常生活中。
那一刻我第一次想到，我恐怕一直在犯那个老掉牙的“心灵投射谬误”-，而且就发生在我的日常生活里，而不是在什么高深抽象的事情上。

我突然第一次意识到，我竟然犯了那个该死的、老掉牙的‘心理投射谬误’，而且不是发生在什么高深抽象的理论里，就发生在我平平无奇的日常生活中。
damned old chestnut：字面意思是“该死的老栗子”，但在口语里通常指“老掉牙的笑话/故事”或者“陈词滥调”。作者用这个词来形容这个谬误，说明它是个非常经典、甚至有点让人听腻了的错误，但自己还是不小心踩坑了。

我这才头一回意识到，自己怕是一直深陷那个老生常谈的思维误区 ——心智投射谬误。而且我并非是在高深抽象的理论思考里犯这个错，反而是在平平无奇的日常生活中时时刻刻都在这么做。


也许问题并不在于我的生产力**格外混沌**；也许只是我在预测它这件事上**格外愚蠢**。 

这就是把愚蠢倒转过来之后呈现出的样子——混沌。某种难以处理、难以把握、难以猜测、让你束手无策的东西。这并不只是一个用来描述人工智能之类高度抽象事物的说法。它在日常生活中同样适用。 

而我们之所以不会想到另一种解释——「我很蠢」——我怀疑**并不是**因为我们把自己看得太高。只是因为我们压根就没有想到自己。我们只是看见了环境中的一个混沌特征。 

所以现在我想到，我的生产力问题也许并不是混沌，而是我自己的愚蠢。 

而这也许会有帮助，也许不会。它当然不能立刻解决问题。说一句「我无知」，并不会让你变得有知识。 

但至少，这和说「事情太混沌了」比起来，是一条不同的路径。
