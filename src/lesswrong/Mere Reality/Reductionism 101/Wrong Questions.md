# 错误的问题

[Wrong Questions](https://www.readthesequences.com/Wrong-Questions)

❦ 

当心智沿着与现实纹理相逆的方向切入时，它就会生成**错误的问题**——这种问题根本不可能在**它自己的前提**下得到回答；只有理解那套会生成「这是个问题」之**感受**的认知算法，才能把它[消解](https://www.readthesequences.com/DissolvingTheQuestion)。 

一个很好的线索是：你甚至都**无法想象**有任何具体而明确的世界状态能够回答这个问题。它甚至看起来都**不可能**被回答。 

举个例子，想想关于荒林中倒树的[标准争论](https://www.readthesequences.com/DisputingDefinitions)。是否存在某种可能的世界状态——某种事态——会对应于「声音」这个词**真正意指**的只是声学振动，或者**真正意指**的只是听觉体验？ 

（「当然有，」其中一方说，「那就是‘声音’意指声学振动的那个事态。」那么，就先把“意指”“代表”等所有类似同义词都[禁用掉](https://www.readthesequences.com/ReplaceTheSymbolWithTheSubstance)，再重新描述一次：什么样的世界状态、什么样的事态，会让一方正确，而另一方错误？） 

如果这还显得太容易，那就拿自由意志来说：无论是在决定论物理学中，还是在带有掷骰子式随机成分的物理学中，究竟会有怎样一种具体事态，能够对应于拥有自由意志？ 

如果**这**也显得太容易，那就问一句：「为什么竟然会有任何东西存在？」然后告诉我，一个令人满意的答案究竟会**是什么样子**。 

而且，不，我也不知道最后那个问题的答案。但根据我过往处理无法回答问题的经验，我**可以**猜到一件事。答案不会是一种宏大而凯旋的第一因。这个问题会随着我对自己的心理算法如何偏离现实运作产生某种洞见而消失；届时我会明白，这个问题从一开始就是错的——问题本身就预设了谬误，内含了那种偏斜。 

神秘存在于心智之中，不在现实之中。如果我对某个现象一无所知，那是关于我心智状态的事实，不是关于那个现象本身的事实。倘若它看起来仿佛根本不存在任何可能的答案，那就更是如此：困惑存在于地图之中，不在疆域之中。**无法回答的**问题，并不是宇宙中魔法介入之处的标记。它们标记的是你的心智偏离现实运行之处。 

这类问题**必须**被消解。你若试图回答它们，坏事就会发生。这几乎必然会生成最糟糕的那类[神秘问题的神秘答案](https://www.readthesequences.com/MysteriousAnswersToMysteriousQuestions)：你似乎为自己的神秘答案想出了很强的论据，但这个「答案」哪怕事后回看，也不能让你作出任何新的预测，而该现象依旧保有它起初那种神圣而不可解的性质。 

比如说，我也可以猜测，第一因之谜的答案是：根本没有任何东西**存在**——「存在」这个整个概念都是假的。但如果你真心相信这一点，你会因此少一些困惑吗？我也不会。 

但**无法回答的**问题最妙的一点在于，至少据我的经验，它们**总是**可以被解决的。伊丽莎白一世女王在自己四十岁生日那天清晨醒来时，脑海里最先闪过的是什么？由于我很容易**想象**出这个问题的答案，我也就很容易看出：我也许永远无法**真正**回答它，因为真正的信息早已湮没在时间之中了。 

另一方面，「为什么竟然会有任何东西存在？」看起来又**如此**绝对不可能，以至于我可以由此推断：我只是在某种意义上陷入了困惑，而真相从绝对意义上说大概并没有那么复杂；等困惑消散之后，我就能看清它。 

如果你从未解决过一个无法回答的问题，这听起来也许会违反直觉；但我向你保证，事情**就是**这样运作的。
