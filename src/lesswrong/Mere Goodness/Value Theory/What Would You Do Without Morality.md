# 如果没有道德，你会怎么做？

[What Would You Do Without Morality?](https://www.readthesequences.com/What-Would-You-Do-Without-Morality)

❦

对于那些说「没有任何东西是真实的」的人，我曾经[回答](https://www.readthesequences.com/QuantumNonRealism)：「很好，那你告诉我，这个什么都不存在的世界到底是怎么运作的？」

假设你突然、而且百分之百确定：没有什么是道德的，也没有什么是对的；所有事都可以做，没有任何事是禁止的。

这当然会是个毁灭性的消息——当然，我并不是在现实里真的告诉你这些。但假设我**真的**这样告诉了你。假设无论你认为自己的道德哲学建立在什么基础上，我都令人信服地把它拆毁了，而且还进一步证明，没有任何东西能取而代之。假设我甚至**证明**了所有效用都等于零。

我知道，你的道德哲学和 [2 + 2 = 4](https://www.readthesequences.com/HowToConvinceMeThatTwoPlusTwoEqualsThree) 一样真实、一样不可证伪。但即便如此，我还是请求你尽最大努力去做这个思想实验，把各种可能性具体地想出来，哪怕它们看上去令人痛苦、毫无意义，或者在逻辑上根本不可能得到任何好的回答。

你还会继续给出租车司机小费吗？你会对自己的伴侣不忠吗？如果一个孩子昏倒在铁轨上，你还会把他拖离铁轨吗？

你还会吃和现在一样的食物吗，还是说你只会吃最便宜的东西，因为没有理由说你**应该**享乐，又或者你会吃非常昂贵的食物，因为没有理由说你**应该**为明天省钱？

你会穿一身黑，写阴郁的诗，把所有利他主义者都斥为傻瓜吗？但你也没有理由说你**应该**这么做——那只不过是一个[缓存想法](https://www.readthesequences.com/CachedThoughts)。你会因为没有理由起床而赖在床上吗？那等到你终于饿了，踉踉跄跄走进厨房之后、吃完东西以后，你又会做什么？

你还会继续读 Overcoming Bias 吗？如果不会，那你又会改读什么？你还会继续努力保持理性吗？如果不会，那你又会怎么想？

闭上眼睛，用你需要的全部时间来回答：

如果没有任何事情是对的，你**会**做什么？
